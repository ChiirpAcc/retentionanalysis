What changed
Win rate now uses the seasonality test output. The recommended model is still the base no-seasonality model, so Q2/Q3/Q4 controls are set to zero. Free months and discount terms are excluded from win rate by product rule.
This avoids forcing the calculator to say free months reduce close rate. Free months still affect churn, LTV, revenue timing, and CAC payback.
Preset interpretation
Recommended default is the practical sales policy. Minimize churn finds the lowest churn within 0–3 free months. Max LTV optimizes already-won customer economics. Max cohort return optimizes revenue from 100 held demos.
Max cohort return includes modeled win rate, so treat it as a sensitivity case, not an automatic pricing policy.
