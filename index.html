Clean calculator logic
Win rate uses onboarding % and MRR. Free months do not affect win rate. Churn uses onboarding % and free months. LTV uses onboarding %, free months, MRR, and projection window.
Contract MRR has no direct churn effect because the churn model excluded it by p-value. MRR still affects revenue, CAC breakeven, LTV:CAC, and modeled win rate.
What this calculator is doing
The model first estimates expected wins from 100 held demos, then applies a monthly churn curve to those won accounts. Monthly revenue declines only because active customers decline.
Max cohort return includes modeled win rate, so use it as a sensitivity case rather than an automatic pricing rule.
